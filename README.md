**Owner-Only Node.js Casino Web App for Termux / Localhost**  

Play **Dice, Slot, and Roulette** with **multi-user support**.  
Only the **Owner** can manage balances and sensitive actions.  

---

## 🚀 Features

- Multi-user, multi-game: Dice, Slot, Roulette  
- Owner-only login & dashboard  
- Localhost live server (Termux-ready)  
- API-ready for future integrations  
- Screenshots & CSS placeholders for UI
